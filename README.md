# Frontend Mentor - Notifications page

![Design preview for the Notifications page coding challenge](preview.jpg)

Esta es la version 0.0.1 de mi solucion para el proyecto de Notification page de Frontend Mentor. Despues de dos años sin codear con este proyecto pretendo retomar el estudio del desarrollo web en general y adaptarme a los estandares y nuevas tecnologias.

Para la estilizacion me base en CSSFlex para organizar los elementos y dejar el proyecto lo mas cercano a los que se observa la imagen.

Para JS decidí mantener un enfoque funcional estableciendo las funciones en el orden en que mis ideas desglosaron el proyecto. El HTML tambien decidi hacerlo con el JS para mantener enfoque funcional y organizado que decidi seguir.

nota: CON EL TIEMPO SE QUE MEJORARE Y QUE EN UNOS MESES VOLVERE A ESTE PROYECTO Y LO ACTUALIZARÉ PARA COMPARAR MI AVANCE.
